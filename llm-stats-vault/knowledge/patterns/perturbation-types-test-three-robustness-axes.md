# Perturbation Types Test Three Robustness Axes

## The Pattern
RQ4 (robustness to prompt variations) uses 75 synthetic tasks derived from 25 base tasks,
each perturbed along three distinct axes.

## Three Perturbation Types

| Type | What Changes | What Stays Same | Tests |
|------|-------------|-----------------|-------|
| `rephrase` | Wording of the prompt | Same inputs, same answers | Prompt sensitivity |
| `numerical` | Input numbers (recomputed ground truth) | Same task type and structure | Generalization ability |
| `semantic` | Real-world framing/context | Same math | Context robustness |

## File Location
`data/synthetic/perturbations.json` — 75 tasks (25 base × 3 types)  
Generated by: `python data/synthetic/build_perturbations.py`

## Perturbation Record Schema
```json
{
  "task_id": "BINOM_FLAT_01_numerical",
  "base_task_id": "BINOM_FLAT_01",
  "perturbation_type": "numerical",
  "task_type": "BINOM_FLAT",
  "tier": 1,
  "difficulty": "basic",
  "prompt": "...",
  "numeric_targets": [...],
  "perturbation_note": "x=5, n=12 instead of x=3, n=10; ..."
}
```

## Running the Synthetic Benchmark
```bash
# All 4 complete models
python -m llm_runner.run_all_tasks \
  --models claude chatgpt deepseek mistral --synthetic

# Gemini (rate-limited)
python -m llm_runner.run_all_tasks \
  --models gemini --synthetic --delay 5

# Filter by perturbation type
python llm_runner/run_all_tasks.py \
  --models claude --synthetic --pert-types numerical
```
